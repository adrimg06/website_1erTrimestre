HTML

HTML (HyperText Markup Language) is the most basic component of the Web. Defines the meaning and structure of web content. In addition to HTML, Other technologies are generally used to describe the appearance/presentation of a web page (CSS) or the functionality/behavior (JavaScript)..


"Hypertext" refers to links that connect web pages to each other, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking to pages created by other people, You become an active participant in the World Wide Web.

HTML examples::

First we will start with the most basic thing, which is a title.

<h1>This is a Heading </h1>


It is used to write text.

<p> Esto es un ejemplo </p>


To link a web page

<a href="https://www.w3schools.com">Visit W3Schools</a>


To change the color of the letter.

<h3 style="color:Tomato;">Hello World</h3>


To put images on the web page.

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
